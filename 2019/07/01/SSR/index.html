Vultr一键搭建SSR

今天看到Vultr的服务器挺便宜的，3.5美元一月，500GB流量，今天就给大家来个教程吧，用来搭建梯子还是很好的
欢迎大家来到我的飞机场，底部有详细介绍
有需要搭建相同网站的可以私信我
购买Vultr服务器
注册地址：Vultr官网（目前注册赠送25美元）

  1. 点击create account开始注册

点击后，你的邮箱会收到注册邮件，点击邮件中的链接确认

  1. 接着跳转到添加充值页面。
    现在PayPal已经支持AliPay了！！点击左侧 Billing 即可看到下图的界面，选择Alipay充值：

这里一个很好的建议是你充值$10美元（好像最低是$10美元），可以用很久

  1. 充值完成后，点击左上角的 Server，然后点击右侧的 + 开始部署属于你自己的服务器

  2. 接下来选择服务器所在节点

服务器节点，我这里选择的是洛杉矶，感觉速度还可以。
你也可以选择Tokyo东京节点，延迟会比较低

  1. 选择系统，这里就默认的Centos7就好

  2. 套餐价格，注意不要选择 $2.5美元一个月的，那个只有IPv6地址，国内很多地方不能用。
    一般选择 $3.5 一个月的就够用了，每个月有500G流量

  3. 剩下的根据自己的需求自己选择，我这里是都不管了，看下左下角的价格，是你选择的套餐价格，就可以直接点击Deploy Now

8.稍等一段时间，就可以部署好了
点击左上角的 Servers 可以看到自己服务器列表，等到看到服务器列表里刚刚购买的服务器是Running 状态就证明服务器部署好了

购买服务器到这一步就结束了，在上图的位置记住你自己的 服务器IP，Username，Password ，准备一键搭建SS

一键搭建SS
其实我是很不想讲怎么用Xshell连接服务器的，这本就是一件很简单的事。所以我不讲了。。。。网上有现成的，麻烦你们看一下。实在不会再来找我。

  1. 下载Xshell（使用普通下载就行）
  2. Xshell连接服务器
  3. 现在假设我们已经使用Xshell连接到服务器了，此刻你应该是root用户

然后开始搭建。
0.一行行执行下面的命令
wget –no-check-certificate -O ss.sh http://tianyu211.coding.me/static/ss.sh
chmod +x ss.sh
./ss.sh 2>&1 | tee ss.log
然后会弹出一些要设置的地方，设置完这些就可以了。我会一步步告诉你要设置什么。
运行上面最后一行之后，会弹出提示像这样：

  1. 设置密码。这一步是让你设置密码，如果不输入就会是默认的 teddysun.com
    输入完回车

2.设置 ss 的连接端口。一般会给你一个随机端口，可以不修改，直接回车，也可以改成你想要的数字

  1. 选择加密模式。输入数字选择你想要的加密模式，这里没有特殊需要的一般直接回车就好

  2. 回车开始搭建。这里提示按任意键开始安装，如果有哪里需要重来，按 Ctrl+C 取消搭建，然后从上面的第 0 步开始。我们回车直接开始搭建

  3. 等待一段时间后，就应该搭建完成了。到时候会弹出来你搭建完成的配置，千万要先保存好配置再关闭窗口！！
    Congratulations,ss-python server install completed!
    Your Server IP : xx.xxx.xxx.xxx // 你的IP
    Your Server Port : 9988 // 你设置的端口
    Your Password : password // 你设置的密码
    Your Encryption Method: aes-256-gcm // 加密方式
    Welcome to visit:https://teddysun.com/342.htmlEnjoy it!
    至此你就彻底搭建完成了！然后打开你的ss连接就好了。
    怎么连接我就不管了，网上一大把。
